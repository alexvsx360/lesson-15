# מצגת אינטראקטיבית - שיעור 15: Git + GitHub

## 🌐 צפייה מקוונת

**[לצפייה במצגת - לחץ כאן](https://alexvsx360.github.io/lesson-15/)**

## 📋 תיאור

מצגת אינטראקטיבית מקיפה המסבירה ניהול גרסאות עם Git ושיתוף קוד עם GitHub - כל מה שצריך לדעת כדי לעבוד בצורה מקצועית.

המצגת כוללת 22 שקופיות עם הסברים מפורטים, דוגמאות קוד, שאלות לדיון כיתתי, ותשובות שניתן להציג בלחיצת כפתור.

- Branches - ענפים

- Pull Requests
- שיתוף פעולה
